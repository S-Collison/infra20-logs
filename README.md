# infra20-logs

Infrasound readings from an INFRA20 meter in and around my property at:

```
1902 Kauffman Ave
Vancouver, WA 98660
```

These are sometimes debilitating and have forced me to move out of my
home for months at a time. Journal entries have correlated to events logged
by this meter. A SerailGhost is logging raw data and those are included
in the LOG.TXT files that are rotated occasionally.

This data can be imported and viewed on [jamaseis](https://www.iris.edu/hq/inclass/software-web-app/jamaseis)

This repositiory is intended to be a storage area for other entities interested
or investigating these events in my home. Feel free to contact me for any additional information.
